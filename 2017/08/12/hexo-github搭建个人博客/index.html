hexo+github搭建个人博客

Posted on 2017-08-12 | In hexo

一.前言

Hexo是一个快速、简洁且高效的博客框架。Hexo使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
使用Hexo+github的缘由是，Hexo纯静态，访问速度比较快，再搭配现在流行的gitHub，可以不需要自己再去弄服务器，只需要买一个域名即可绑定。
其实网上的教程也挺多的，本来自己不打算写，但是网上的看着要么不够全面，要么比较繁琐复杂，所以决定自己写点东西。
