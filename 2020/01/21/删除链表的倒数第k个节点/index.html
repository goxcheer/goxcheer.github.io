删除链表的倒数第k个节点

题目描述

给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

示例：

给定一个链表: 1->2->3->4->5, 和 n = 2.

当删除了倒数第二个节点后，链表变为 1->2->3->5.

说明：

给定的 n 保证是有效的。

解答

两次遍历

这个题目最容易想到的是两次遍历，第一次遍历记录链表的长度len，第二次遍历到
len - n 次进行删除操作即可。

一次遍历

采用双指针辅助，第一个指针先走n+1步，然后两个指针以恒定的距离n+1分别走直到第一个指针到达链表的末端，然后删除第二个指针的next即可。

为了代码可读性强，可以采用哑节点作为头结点辅助.
