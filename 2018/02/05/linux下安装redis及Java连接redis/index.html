linux安装redis

安装gcc编译软件

　redis是用c编写的，在下载之后需要使用gcc编译之后才能安装，首先需要下载gcc。

下载redis安装包，并解压编译

进入redis官网,传送门
复制下载链接地址 http://download.redis.io/releases/redis-4.0.11.tar.gz

然后编译。

检查是否编译成功

安装redis

指定安装到/usr/local/redis/

安装完成后,进入目录 cd redis
里面有两个文件目录，进入bin,里面的redis-server就是redis启动命令.
然后bin/redis-server,启动成功的界面：

出现这个图形就是redis启动成功。
