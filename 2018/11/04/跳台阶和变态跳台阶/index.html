跳台阶和变态跳台阶

题目描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。

解题思路

这种问题我们不妨先用数学归纳法试试，
n = 1 ,1
n = 2 ,2
n = 3 ,3
n = 4, 5
也就是n >= 2, fn = f(n-1)+f(n-2)，这就是斐波拉契数列。
换个方向思考，其实也就是我每次跳到n级台阶，要么就是从n-1级台阶跳上来的，要么就是从n-2级台阶跳上来的，所以就是到n-1，n-2台阶方法之和。

题目描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

解题思路

有了前面的基础，这个现在看起来也就不难了，唯一需要注意的是每个fn都多了一种可能是从开始节点直接一次跳上来，所以需要+1.

java代码实现

public int JumpFloorII(int target) {
if (target == 0){
return 0;
}
int num = 1;
for (int i = 1 ; i < target; i++){
num = num + JumpFloorII(target-i);
}
return num;
}

题目描述

输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。

解题思路

二进制位数中，不是0就是1，假设一个二进制数n，我们可以将n和n-1进行与运算，这样每次与运算一次之后，1的个数都会减一，直到最后为0.
