从尾到头打印链表

Posted on 2018-11-12 In 算法 , 链表

输入一个链表，按链表从尾到头的顺序返回一个ArrayList.

题目描述

输入一个链表，按链表从尾到头的顺序返回一个ArrayList。

public class ListNode {
int val;
ListNode next = null;
ListNode(int val) {
this.val = val;
}
}

解题思路

考虑到链表是单向的，我们从头到尾遍历，但是需求结果是从尾到头，我们很容易联想到用LIFO的栈来辅助。遍历将每一个结果入栈，出栈顺序即从尾到头。还有一种思路是，我们将每次遍历的子序列进行递归，递归返回的顺序就是从尾到头的顺序。

java代码实现

  • 栈辅助

    public ArrayList printListFromTailToHead(ListNode listNode) {

    ArrayList<Integer>list = new ArrayList<Integer>();
Stack<Integer>stack = new Stack<>();
while (listNode!=null){
    stack.push(listNode.val);
    listNode = listNode.next;
}
while (!stack.isEmpty()){
    list.add(stack.pop());
}
return list;

    }

  • 递归遍历

public ArrayList printListFromTailToHead(ListNode listNode) {
ArrayListlist = new ArrayList();
if (listNode != null){
list = printListFromTailToHead(listNode.next);
list.add(listNode.val);
}
return list;
}
